
# World

__Make it simple to work with locations and data about the world __

It should be easy to do things like get an list of US States, Canadian
Provinces, or to determine if a postal code or phone number is valid.

    - Add Editor-specific files to your .gitignore ( .swp for Vim, .idea for PHPStorm, etc. )
